# TecnoStock - Sistema de Control de Inventario y Ventas

Este proyecto es una aplicación de consola interactiva desarrollada en C++ diseñada para la gestión simplificada de inventarios y análisis de métricas comerciales en una tienda de tecnología.

## 🚀 Funcionalidades Principales
1. **Inicialización Automática:** El sistema arranca con datos cargados por defecto para facilitar pruebas rápidas.
2. **Registro y Validación:** Permite sobrescribir los 5 productos controlando que no se ingresen precios ni stock negativos.
3. **Reporte Tabulado:** Muestra de forma limpia e higiénica una tabla con el valor total invertido por ítem y el acumulado en bodega.
4. **Módulo de Estadísticas:** Calcula de forma automática promedios de precios, totales de unidades y detecta dinámicamente los productos máximos y mínimos en costo.
